Inserindo Textos

A princípio é bem simples inserir textos utilizando HTML, com pouco esforço já é possível colocar um título, subtítulo, parágrafos e entre outras coisas no seu texto. Vamos começar com o título, veja como é fácil colocar um título na sua página.

Títulos

Para colocar um título chamativo na sua página é necessário inserir apenas a tag '< h1 >' no seu código HTML e aparece um título como mágica na sua página. Ele será, é claro, um texto maior do que os demais presentes.

Você pode inserir títulos, subtítulos e assim vai, apenas incrementando o número depois do 'h', assim podemos criar um subtítulo com um '< h2 >', por exemplo.

Parágrafos

Para inserir um parágrafo é mais simples ainda, basta inserir a tag '< p >' com o texto da sua preferência que você terá um parágrafo na sua página.

Modificando Palavras

(Voltar)

Podemos também inserir modificações nas palavras inseridas no seu texto, como você faz no seu Word inserindo palavras sublinhadas, em negrito, itálico, etc. Podemos inserí-las também utilizando algumas das tags HTML abaixo:

  • < u > para inserir uma palavra sublinhada
  • < mark > para inserir uma palavra com um fundo estilo marca texto
  • < i > para inserir uma palavra em itálico
  • < strong > para inserir uma palavra em negrito
  • < sub > para inserir uma palavra na parte inferior em relação as outras r
  • < sup > para inserir uma palavra na parte superior em relação as outras r
  • < del > para inserir uma palavra com um risco no meio, geralmente assimilada a exclusão Deletado
  • < abbr > para inserir uma explicação para uma abreviação ONU

Listas

(Voltar)

A inserção de listas no HTML é bem simples, mas antes de começar a inserir, pergunte-se se você quer fazer uma lista ordenada ou não ordenada. Esses dois tipos de lista tem uma tag para cada. Veja a seguir a tag de cada um tipo de lista:

Lista Ordenada

Para inserir uma lista ordenada basta utilizar a tag '< ol >' que representará a lista ordenada e a tag '< li >' que representará o conteúdo da lista. Veja abaixo como que fica o estilo da lista ordenada:

  2. Ordenada

Lista Não Ordenada

Agora para inserir uma lista ordenada utilizamos a tag '< ul >' que representa a lista não ordenada e como a lista ordenada, utilizamos a tag '< li >' para representar o conteúdo da lista. Veja abaixo como que fica o estilo da lista não ordenada:

Links

(Voltar)

Podemos utilizar os links para realizar diversas funções como, por exemplo:

  • Redirecionar o usuário para outro site
  • Redirecionar o usuário para outra página do seu site
  • Redirecionar o usuário para um conteúdo na mesma página do link

Tudo isso com a seguinte sintaxe simples, veja abaixo:

< a href="caminho">texto link< /a >

Dica

Caso você queira que o redirecionamento abra uma nova janela no navegador, basta inserir o atributo 'target="_blank"' na sua tag '< a >'.
